# Descriptive Analysis and Model Validation

Baseline descriptive statistics and validation of the thermal model.

## Notebooks

| Notebook | Purpose |
|----------|---------|
| `description_stock.ipynb` | Building stock composition, thermal transmittance, and consumption statistics |
| `static_validation.ipynb` | Thermal function validation (DPE A-G range), consumption by construction period, static cost-efficiency |
| `climate.ipynb` | Climate variation impact on heating demand (2010-2019) |
| `hourly_consumption.ipynb` | Hourly heating demand profiles with smoothing |
